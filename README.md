# 🇺🇦 Olympic Ukraine API

A professional web application designed to manage and visualize data about Ukrainian Olympians. This project was developed as part of a university laboratory series to demonstrate proficiency in Node.js, REST API architecture, and MongoDB integration.

## 🚀 Key Features
* **Data Dashboards**: View comprehensive lists of athletes, sports, and Olympic Games history in structured, styled tables.
* **Athlete Search**: Quickly find specific athlete details using their unique ID through a dedicated search interface.
* **Performance Comparison**: Side-by-side comparison of athlete rankings with interactive charts powered by **Chart.js**.
* **Localized UI**: A user-friendly interface styled with Ukraine's national colors (Blue & Yellow).

## 🛠 Tech Stack
* **Runtime**: Node.js (v24.14.0+)
* **Backend Framework**: Express.js
* **Database**: MongoDB with Mongoose ODM
* **Visualization**: Chart.js (for ranking analytics)
* **Development Tools**: VS Code, MongoDB Compass, Git
